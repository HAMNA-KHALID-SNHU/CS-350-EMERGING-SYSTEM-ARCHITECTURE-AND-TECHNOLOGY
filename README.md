## Repository Overview

This repository contains my work from CS-350: Emerging Systems Architecture and Technology.
The projects focus on building embedded systems using a Raspberry Pi, integrating hardware components, and developing software to control real-time system behavior.

The final project is a Smart Thermostat System that reads environmental data, processes user input, and controls system output using a structured state machine.

### Included Artifacts

* Thermostat.py → Final Smart Thermostat System
* Milestone3.py → Morse Code LED State Machine
* State_Diagram.pdf → System State Machine Design
* Reports/ → Project documentation and analysis

---
### Project Highlights

* Real-time temperature monitoring system
* State machine implementation (OFF / HEAT / COOL)
* Hardware integration using:
* LEDs (status indicators)
  * Push buttons (user input)
  * LCD display (system output)
  * Temperature sensor (environment data)
* UART communication for simulated data transmission
